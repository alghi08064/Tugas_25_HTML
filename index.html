    • Mengenal Semantic Element Di HTML5

    • Article

      • Ketika ingin membuat layout untuk menampilkan blok artikel-artikel yang ada di website kita, tag article bisa kita gunakan dengan element header dan footer.
      Karena tag article memiliki block type, artinya element ini memiliki lebar browser jika kita tidak menentukan lebarnya.

    • Aside

      • Element ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside di bagian bawah postiongan untuk menjelaskan identitas penulis artikel.
      karena aside memiliki type/sifat block, maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
